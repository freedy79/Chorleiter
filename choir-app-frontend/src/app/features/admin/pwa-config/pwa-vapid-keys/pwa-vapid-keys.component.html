Was sind Push-Benachrichtigungen?

Push-Benachrichtigungen informieren Chor-Mitglieder über neue Chat-Nachrichten und Beiträge, auch wenn die App gerade nicht geöffnet ist. Sie werden als kleine Meldungen auf dem Computer oder Smartphone angezeigt – ähnlich wie bei WhatsApp oder E-Mail.

Wie funktioniert das?

  1. VAPID-Schlüssel generieren: Klicken Sie unten auf „Schlüssel generieren". Das erstellt automatisch die nötigen kryptografischen Schlüssel.
  2. Subject eintragen: Geben Sie Ihre E-Mail-Adresse als Kontakt ein (Format: mailto:ihre@email.de).
  3. Backend neu starten: Nach dem Speichern muss der Server einmal neu gestartet werden, damit die Schlüssel aktiv werden.
  4. Benutzer aktivieren: Jedes Chor-Mitglied muss im eigenen Browser die Benachrichtigungen erlauben, wenn es dazu aufgefordert wird.

Auf welchen Geräten funktioniert es?

  • Desktop: Chrome, Edge, Firefox – sofort nach Erlaubnis
  • Android: Chrome, Edge, Firefox – sofort nach Erlaubnis
  • iPhone/iPad: Safari ab iOS 16.4 – nur wenn die App zum Homescreen hinzugefügt wurde

Was passiert, wenn ich neue Schlüssel generiere?

Achtung: Wenn Sie neue Schlüssel generieren, werden alle bestehenden Push-Abonnements ungültig. Die Chor-Mitglieder müssen danach erneut die Benachrichtigungen aktivieren. Generieren Sie neue Schlüssel nur bei der Ersteinrichtung oder wenn es Sicherheitsprobleme gibt.

Alternative: CLI-Einrichtung

Fortgeschrittene Administratoren können die Schlüssel auch über die Kommandozeile generieren:

cd choir-app-backend && npm run generate-vapid-keys -- --save --subject mailto:ihre@email.de
