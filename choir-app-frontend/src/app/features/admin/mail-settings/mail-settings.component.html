Mail-Domain Tragen Sie als erstes die Domain ein, über die Sie versenden möchten. SMTP-Server Benutzername (kurzname) Der SMTP-Benutzername ist in der Regel gleichzeitig Ihre E-Mail-Adresse. Passwort Tragen Sie außerdem das passende Passwort zu Ihrem E-Mail-Postfach ein. Verschlüsselung (TLS/SSL) Port Als letzte Angabe müssen Sie nun einen Port mit passender Verschlüsselung eintragen. Bitte wählen Sie eine Verschlüsselung aus! Wir empfehlen Port 587.
